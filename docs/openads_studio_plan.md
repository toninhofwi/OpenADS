# OpenADS Studio — design plan (retired)

This page used to describe a desktop Dear-ImGui Studio. It was
**superseded** before any of it shipped — OpenADS Studio is now a
**web console**, served either by `openads_serverd` (Remote Server
mode) or in-process by `ace64.dll` / `ace32.dll` (LocalServer mode,
since v1.0.0-rc9).

See the current docs:

- **[Studio guide (EN)](en/studio-guide/)**
- **[Guía de Studio (ES)](es/guia-studio/)**
- **[Guia do Studio (PT)](pt/guia-studio/)**

The historical design discussion that lived here has been removed to
avoid confusing readers about which Studio actually exists. The
git history (`git log -- docs/openads_studio_plan.md`) preserves
the original document for archaeology.
